IslandServer

This item only works when running on the server. Server

This class holds all island data for each spawned island. It should be the only way you're interacting with the data.

DANGER

Changing any of the island's properties directly may lead to unexpected behavior. All of the properties should be only read and changed only through the island's methods.

Properties​

ID​

This item is read only and cannot be modified. Read Only
</>
IslandServer.ID: number

The identification property of the island.

Owner​

This item is read only and cannot be modified. Read Only
</>
IslandServer.Owner: Player | {UserId: number}

The owner of the island. If the island is not owned by a player, this property is equal to the table {UserId = 0}.

OwnedByPlayer​

This item is read only and cannot be modified. Read Only
</>
IslandServer.OwnedByPlayer: boolean

True when the the island is owned by a player and false otherwise.

Loaded​

This item is read only and cannot be modified. Read Only
</>
IslandServer.Loaded: boolean

True when the island is spawned and prepared (with everything loaded) and false otherwise. Turns false after the island is destroyed.

TIP

Useful to check when the island needs to be fully loaded or fully destroyed before continuing.

Destroyed​

This item is read only and cannot be modified. Read Only
</>
IslandServer.Destroyed: boolean

Turns true right before the island is destroyed.

TIP

Useful to check when the island needs to be fully working or in the process of loading before continuing. Good to pair with IslandServer.Loaded.

Perms​

This item is read only and cannot be modified. Read Only
unreleased
</>
IslandServer.Perms: {Player}

Holds the players that have perms to edit the island. Contains the owner.

WARNING

This property isn't implemented yet. It currently has no use.

Model​

This item is read only and cannot be modified. Read Only
</>
IslandServer.Model: IslandModel?

Reference to the workspace model of the island, if one exists (if the island is spawned).

HQConfig​

This item is read only and cannot be modified. Read Only
</>
IslandServer.HQConfig: ConfigTypes.HeadquartersConfig

The config of the island at its current level.

This item is read only and cannot be modified. Read Only

A table of the NodeServer instances that the island contains.

Roads​

Trucks​

Packages​

This item is read only and cannot be modified. Read Only
</>

A table of the PackageServer instances that the island contains.

Cities​

This item is read only and cannot be modified. Read Only
</>
IslandServer.Cities: {[ID]: CityServer}

A table of the CityServer instances that the island contains.

HQNode​

This item is read only and cannot be modified. Read Only
</>
IslandServer.HQNode: NodeServer?

The node that is associated with the island's HQ. This property is nil until the island spawns.

DestroyJanitor​

This item is read only and cannot be modified. Read Only
</>
IslandServer.DestroyJanitor: Janitor

Gets cleaned up once the island is being destroyed.

IslandServer.DestroyJanitor:Add(RunService.Heartbeat:Connect(function(dt)
	-- code
end), "Disconnect")
TIP

:Add() connections and instances that need to be cleaned up (destroyed/disconnected) when the island gets destroyed to battle memory leaks.

INFO

The first argument is the item that needs to be cleaned up and the second argument is the method invoked for cleaning up. For instances and default roblox connections, the second argument can be omitted and determined automatically.

HQLevel​

This item is read only and cannot be modified. Read Only
</>
IslandServer.HQLevel: number

The current level of the HQ.

Cash​

This item is read only and cannot be modified. Read Only
</>
IslandServer.Cash: number

How much cash the island currently has.

TotalCash​

This item is read only and cannot be modified. Read Only
</>
IslandServer.TotalCash: number

How much cash the island earned in total over its lifetime.

TotalPackages​

This item is read only and cannot be modified. Read Only
</>
IslandServer.TotalPackages: number

How many packages were delivered on this island.

TimePlayed​

This item is read only and cannot be modified. Read Only
</>
IslandServer.TimePlayed: number

For how long the island is spawned.
