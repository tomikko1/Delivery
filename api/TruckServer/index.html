TruckServer

This item only works when running on the server. Server

This class holds the data of every truck on the island.

DANGER

Changing any of the object's properties directly may lead to unexpected behavior. All of the properties should be only read and changed only through the object's methods.

Properties​

ID​

This item is read only and cannot be modified. Read Only
</>
TruckServer.ID: number

The identification property of the truck.

Island​

This item is read only and cannot be modified. Read Only
</>
TruckServer.Island: IslandServer

The island the truck is associated with.

TIP

Use this property if you have a truck and you need to interact with the island the truck is on.

Type​

This item is read only and cannot be modified. Read Only

This item is read only and cannot be modified. Read Only
</>
TruckServer.Config: ConfigTypes.TruckConfig

The truck's config.

CarryingWeight​

This item is read only and cannot be modified. Read Only

TruckServer.CarryingWeight: number

What weight the truck is carrying.

Packages​

This item is read only and cannot be modified. Read Only
</>
TruckServer.Packages: {[ID]: PackageServer}

A list of the packages the truck is carrying.

Traveling​

This item is read only and cannot be modified. Read Only
</>
TruckServer.Traveling: boolean

True when the truck is traveling, false otherwise.

CurrentNode​

This item is read only and cannot be modified. Read Only
unreleased
</>
TruckServer.CurrentNode: NodeServer?

The last node the truck has been to.

DANGER

This property is not being used yet and therefore its behavior isn't thoroughly tested. Changing it shouldn't have any implemented behavior.

DestinationReached:​

This item is read only and cannot be modified. Read Only
</>
TruckServer.DestinationReached:: Signal<NodeServer>

This signal fires every time a truck has reached its destination, whether that was a city the truck delivered packages to or the HQ.

LoadPackages​

</>

Picks the best available packages from the HQ and loads the truck with them. This method prioritizes grouping packages from the same city, to minimize the truck's travel distance.
