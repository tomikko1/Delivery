PackageServer

This item only works when running on the server. Server

This class holds the data of every package on the island.

DANGER

Changing any of the object's properties directly may lead to unexpected behavior. All of the properties should be only read and changed only through the object's methods.

Properties​

ID​

This item is read only and cannot be modified. Read Only
</>
PackageServer.ID: number

The identification property of the package.

Island​

This item is read only and cannot be modified. Read Only
</>
PackageServer.Island: IslandServer

The island the package is associated with.

TIP

Use this property if you have a package and you need to interact with the island the package is on.

Type​

This item is read only and cannot be modified. Read Only
</>
PackageServer.Type: ConfigTypes.PackageType

What type the package is.

Config​

This item is read only and cannot be modified. Read Only
</>
PackageServer.Config: ConfigTypes.PackageConfig

The package's config.

Price​

This item is read only and cannot be modified. Read Only
</>
PackageServer.Price: number

The price at which the package will be sold. This is equal to the price in the package's config + some random deviation by 10% at most.

Destination​

This item is read only and cannot be modified. Read Only
</>
PackageServer.Destination: CityServer

The city that spawned the package and is requesting it to be delivered.

Departed​

This item is read only and cannot be modified. Read Only
</>
PackageServer.Departed: boolean

True when the package is out of HQ, false otherwise.
